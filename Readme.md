# README: Y2K Retro Portfolio

## 📋 Project Overview
A retro-futuristic portfolio website for a Computer Engineering student with Y2K aesthetics and a built-in maze game.

## ✨ Features
- **Portfolio Sections**: About, Skills, Projects, Experience, Contact
- **Y2K Design**: CRT effects, neon colors, glitch animations, scanlines
- **Interactive Game**: "Bug Collector" - maze game with power-ups
- **Responsive**: Works on desktop and mobile

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Font Awesome, Google Fonts

## 🎮 Game Controls
- **Arrow Keys/WASD**: Move
- **Spacebar**: Attack bugs / Use power-up
- **1,2,3**: Select power-ups
- **R**: Restart level

## 📁 Files
- `index.html` - Main page
- `style.css` - All styling
- `script.js` - Game logic & interactivity

## 🚀 Quick Start
1. Download all 3 files in same folder
2. Open `index.html` in browser
3. Navigate sections via top buttons
4. Click "MINI-GAME" to play
